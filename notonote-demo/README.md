# NotoNote Demo

Static prototype for the NotoNote multimodal meeting notes demo.

## Pages

- `index.html`: workbench and phone app preview.
- `index.html?view=phone`: phone-only display mode.
- `index.html?view=phone&id=<result_id>&sb_url=<supabase_url>&sb_key=<publishable_key>`: load a saved Supabase result.

## Notes

Do not commit database passwords, direct connection strings, Gemini API keys, or Supabase service role keys.
